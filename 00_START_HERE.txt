⚡ COMMANDES RAPIDES
═══════════════════════════════════════════════════════════════════════════════

  Backend:
    npm run dev          Développement avec hot-reload
    npm start            Production
    npm test             Lancer tous les tests
    npm run test:watch   Tests avec reload

  Frontend:
    npm start            Développement
    npm run build        Build production
    npm test             Tests

  Docker:
    docker-compose up -d            Démarrer tous services
    docker-compose down             Arrêter
    docker-compose logs -f          Afficher logs
    docker-compose ps               Statut services

  Database:
    node backend/db-init.js         Initialiser la DB
    psql -U edumanager_user ...     Se connecter
